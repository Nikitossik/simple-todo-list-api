# Installation
1) Clone this repository:
```bash
git clone https://github.com/Nikitossik/simple-todo-list-api
```
2) Go to the root directory:
```bash
cd simple-todo-list-api
```
3) Install requirements:
```bash
pip install -r requirements.txt
```
4) Run application:
```bash
flask --app main run
```
